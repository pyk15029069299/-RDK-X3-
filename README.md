# -RDK-X3-
本项目基于地平线RDK X3开发平台搭建车载智能辅助系统，主要实现疲劳驾驶监测与道路目标检测测距功能。依托RDK X3内置BPU算力，将YOLOv5目标检测模型与EAR眼部特征疲劳判别算法部署至边缘端，在Linux环境下搭建低时延实时感知-逻辑决策-声光交互一体化系统。系统采用双摄像头采集画面，MIPI与USB相机分别采集驾驶员面部画面与前方道路画面；利用YOLOv5识别道路车辆、行人并测算相对距离，借助EAR算法分析眼部开合程度判定疲劳等级；当驾驶员进入疲劳状态时，设备通过声卡播放语音预警，同时将目标类型、置信度、测距结果可视化展示于GUI交互界面，实现行车安全实时辅助提醒。
